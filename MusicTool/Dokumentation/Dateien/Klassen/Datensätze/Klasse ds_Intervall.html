Klasse: ds_Intervall

 

ABGELEITET VON: bcl_Basisdata2
Diese Klasse ist ein Datensatz für die Datenbank DB_Intervalle.
Der Datensatz kann von allen Klassen ausgewertet werden. Er kann auch über die Basisklasse bcl_Basisdata2 angesprochen werden.
Als Typ wird der Typ Intervalldaten gespeichert.

Aufruf: ds_Intervall ( name , altName , id , basisIntervall )


öffentliche Schnittstellen:

    Eingabe:
    • Konstruktor( name , altName , id , basisIntervall )
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
        Die Parameter
        • name ,
        • altName ,
        • id (als INT) und
        • typ = dType.Intervalldaten
        werden an die Basisklasse bcl_Basisdata übergeben.
      • Folgende Daten werden gespeichert:
        1. id ==> ID des Intervalls in der Datenbank DB_Intervalle
          ==> TYP: eIntervall
        2. basisIntervall ==> Basis-Intervall
          ==> TYP: eBasisIntervall

    -------------------------------------------------------------

    Ausgabe:
    • Eigenschaften:
      1. IntervallName
        gibt den Intervall-Namen aus
        ==> TYP: STRING
      2. AltIntervallName
        gibt den alternativen Intervall-Namen aus
        ==> TYP: STRING
      3. IntervallID
        gibt die IntervallID aus
        ==> TYP: eIntervall
      4. BasisIntervall
        gibt das BasisIntervall aus
        ==> TYP: eBasisIntervall
      5. IntBasisIntervall
        gibt das BasisIntervall als INT aus
        ==> TYP: INT

      6. geerbte Eigenschaften:
        1. ID
          gibt die ID des Objekts aus
          ==> TYP: INT
        2. Typ
          gibt den Datentyp des Objekts aus
          ==> TYP: dType
        3. AltName
          gibt den alternativen Namen des Objekts aus
          ==> TYP: STRING
        4. Name
          gibt den Namen des Objekts aus
          ==> TYP: STRING
