Klasse: DB_Intervalle

 

Diese Klasse ist eine (STATISCHE) Datenbank aller bekannten Intervalle. Sie kann also NICHT instanziiert werden.


öffentliche Schnittstellen:

    Eingabe:
    • Konstruktor() (wird automatisch ausgeführt)
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
        Die Daten werden durch private Methoden in die Datenbank gespeichert.

    -------------------------------------------------------------

    Ausgabe:
    • Eigenschaften:
      1. Anzahl
        gibt die Anzahl der Datensätze aus
        ==> TYP: INT
      2. Name
        gibt den Namen der Liste aus
        ==> TYP: STRING
      3. Typ
        gibt den Daten-Typ der Liste aus
        ==> TYP: dType

    • Methoden:
      1. get( id )
        gibt einen Intervall-Datensatz aus
        Parameter:
        1. id
          ID des Intervall
          ==> TYP: eIntervall
        Rückgabe:
          Intervall-Datensatz
          ==> TYP: ds_Intervall

      2. get( id )
        gibt einen Intervall-Datensatz aus
        Parameter:
        1. id
          ID des Intervall
          ==> TYP: INT
        Rückgabe:
          Intervall-Datensatz
          ==> TYP: ds_Intervall
