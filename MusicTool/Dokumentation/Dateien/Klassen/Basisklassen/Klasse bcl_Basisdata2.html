Klasse: bcl_Basisdata2

 

Diese Klasse dient als Basisklasse für alle Datensätze mit einem zusätzlichen alternativen Namen.
Sie speichert den Namen und den alternativen Namen der Instanz, die ID und den Datentyp.

Aufruf: bcl_Basisdata2 ( name , altName , id , typ )


öffentliche Schnittstellen:

    Eingabe:
    • Konstruktor( name , altName , id , typ )
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
      • Folgende Daten werden gespeichert:
        1. name
          ==> Name des Objekts
          ==> TYP: STRING
        2. altName
          ==> alternativer Name des Objekts
          ==> TYP: STRING
        3. id
          ==> ID des Objekts
          ==> TYP: INT
        4. typ
          ==> Datentyp des Objekts
          ==> TYP: dType

    -------------------------------------------------------------
    Ausgabe:
    • Eigenschaften:
      1. ID
        gibt die ID des Objekts aus
        ==> TYP: INT
      2. Typ
        gibt den Datentyp des Objekts aus
        ==> TYP: dType
      3. AltName
        gibt den alternativen Namen des Objekts aus
        ==> TYP: STRING
      4. Name
        gibt den Namen des Objekts aus
        ==> TYP: STRING
