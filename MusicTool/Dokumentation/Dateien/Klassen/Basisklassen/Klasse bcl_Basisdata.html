Klasse: bcl_Basisdata

 

Diese Klasse ist zum Vererben gedacht.
Sie speichert den Namen der Instanz, die ID und den Typ der Daten.

Aufruf: bcl_Basisdata ( name, typ, id )


öffentliche Schnittstellen:

    Eingabe:
    • Konstruktor( name, typ, id )
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
      • Folgende Daten werden gespeichert:
        1. name
          ==> Name der Instanz
          ==> TYP: STRING
        2. typ
          ==> Typ der Instanz
          ==> TYP: dType
        3. id
          ==> ID der Instanz
          ==> TYP: INT
    -------------------------------------------------------------
    Ausgabe:
    • Eigenschaften:
      1. Typ
        gibt den Typ des Datensatzes aus
        ==> TYP: dType
      2. ID
        gibt die ID der Instanz aus
        ==> TYP: INT
      3. Name
        gibt den Namen des Objekts aus
        ==> TYP: STRING




Interne Struktur:

Die VersionInfo kann nur durch eine spezielle Klasse angezeigt werden.
  • Attribute:
    1. _name
      Name der Instanz
      ==> TYP: STRING
    2. _id

      ==> TYP: INT
    3. _typ

      ==> TYP: dType
