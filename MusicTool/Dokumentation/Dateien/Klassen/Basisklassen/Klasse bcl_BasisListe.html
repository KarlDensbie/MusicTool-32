Klasse: bcl_BasisListe<T>

 

Diese Klasse dient als Basisklasse für alle Listen von Objekten.
Der Datentyp wird bei der Instantiierung festgelegt.

Aufruf: bcl_BasisListe ( name , typ , id )


öffentliche Schnittstellen:

    Eingabe:
    • Konstruktor( name , typ , id )
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
      • Folgende Daten werden gespeichert:
        1. name
          ==> Name der Instanz
          ==> TYP: STRING
        2. typ ==> Typ der Instanz
          ==> TYP: dType
        3. id ==> ID der Instanz
          ==> TYP: INT

    • Methoden:
      1. ADD( element )
        fügt ein Element an die Liste an.
        Parameter:
        1. element
          neues Element der Liste
          ==> TYP: T (generisch)
        Rückgabe:
          KEINE

    -------------------------------------------------------------

    Ausgabe:
    • Eigenschaften:
      1. Typ
        gibt den Typ der Liste aus
        ==> TYP: dType
      2. Anzahl
        gibt die Anzahl der Listen-Elemente aus
        ==> TYP: INT
      3. Liste
        gibt die Liste der Elemente aus
        ==> TYP: List<T>
      4. Name
        gibt den Namen der Liste aus
        ==> TYP: STRING
      5. ID
        gibt die ID der Liste aus
        ==> TYP: INT

    • Methoden:
      1. getElement( id )
        gibt ein Element der Liste aus
        Parameter:
        1. id
          ID der Liste
          ==> TYP: INT
        Rückgabe:
          Listen-Element
          ==> TYP: <T> (generisch)
