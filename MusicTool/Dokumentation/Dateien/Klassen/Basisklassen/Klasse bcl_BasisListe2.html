Klasse: bcl_BasisListe2<T>

 

Diese Klasse dient als Basisklasse für alle Listen von Objekten mit zusätzlichem alternativen Namen.
Der Datentyp wird bei der Instantiierung festgelegt.

Aufruf: bcl_BasisListe2<DATENTYP> ( name , altName , typ , id )


öffentliche Schnittstellen:

    Eingabe:

    • Konstruktor( name , altName , typ , id )
        • EINZIGE Möglichkeit zur Speicherung der Objektdaten:
      • Folgende Daten werden gespeichert:
        1. name
          ==> Name der Liste
          ==> TYP: STRING
        2. altName
          ==> alternativer Name der Liste
          ==> TYP: STRING
        3. typ
          ==> Typ der Liste
          ==> TYP: dType
        4. id
          ==> ID der Liste
          ==> TYP: INT

    • Methoden:
      1. ADD( element )
        fügt ein Element an die Liste an.
        Parameter:
        1. element
          neues Element der Liste
          ==> TYP: T (generisch)
        Rückgabe:
          KEINE

    -------------------------------------------------------------

    Ausgabe:
    • Eigenschaften:
      1. Typ
        gibt den Typ der Liste aus
        ==> TYP: dType
      2. AltName
        gibt den alternativen Namen der Liste aus
        ==> TYP: STRING
      3. Name
        gibt den Namen der Liste aus
        ==> TYP: STRING
      4. ID
        gibt die ID der Liste aus
        ==> TYP: INT
      5. Liste
        gibt die Liste der Elemente aus
        ==> TYP: List<T>
      6. Anzahl
        gibt die Anzahl der Listen-Elemente aus
        ==> TYP: INT

    • Methoden:
      1. getElement( id )
        gibt ein Element der Liste aus
        Parameter:
        1. id
          ID des Listen-Elements
          ==> TYP: INT
        Rückgabe:
          Listen-Element
          ==> TYP: <T> (generisch)
